About This Demo

This page shows two types of login:

  • 🚨 Insecure: Password checked directly in code.
  • ✅ Secure: Password is hashed before checking.

Key Learnings

  • ❌ Don’t hardcode usernames/passwords.
  • ❌ Don’t use plaintext checks.
  • ✅ Use hashing (bcrypt, Argon2, etc.).
  • ✅ Real apps should check passwords on the server.
